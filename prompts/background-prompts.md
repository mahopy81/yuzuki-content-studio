# GPT Image 2 背景生成プロンプト

Yuzuki Content StudioのInstagramカルーセルでは、日本語テキストはHTML/CSSで描画します。
画像生成AIには文字を入れさせず、背景・装飾・紙素材だけを作らせてください。

## Low: 案出し用 背景

```text
Create a soft pastel pink grid-paper background for a Japanese Instagram carousel.
Cute, feminine, minimal, warm, gentle, clean.
Add subtle hand-drawn hearts, clips, small arrows, and paper-like texture.
No text, no letters, no numbers, no logo.
Vertical 4:5 composition, suitable for overlaying Japanese text later.
```

## Medium: 本番用 背景

```text
Create a polished pastel pink grid-paper background for a Japanese Instagram carousel template.
Use soft rose pink, dusty pink, white, and very light gray.
The design should feel cute, feminine, friendly, beginner-friendly, and readable.
Add very subtle paper texture, small hand-drawn hearts, paper clips, arrows, and soft shadows.
Keep the center and upper areas clean for large Japanese headings and text cards.
No text, no letters, no numbers, no logo, no watermark.
Vertical 4:5 composition, 1080 by 1350 px.
```

## Medium: 装飾だけ

```text
Create a transparent-background set of cute pastel pink Instagram carousel decorations.
Include hand-drawn hearts, arrows, check marks, paper clips, washi tape, small speech bubbles, and sticker-like shapes.
Soft rose pink and dusty pink.
No text, no letters, no numbers, no logo.
PNG elements, clean edges, suitable for overlaying on HTML text layouts.
```

## Medium: 紙カード素材

```text
Create soft white translucent paper cards for a pastel pink Instagram carousel.
Rounded corners, very light shadow, subtle paper texture, gentle feminine style.
Include several card shapes: large text card, small label card, checklist card, speech bubble card.
No text, no letters, no numbers, no logo.
Transparent background, suitable for placing Japanese HTML text on top.
```

## NG指定

```text
Do not generate Japanese text.
Do not generate English text.
Do not generate fake letters, numbers, signatures, logos, or watermarks.
The final text will be added later with HTML and CSS.
```
